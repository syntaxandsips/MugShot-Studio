# Color Documentation

## Primary Colors
- **Primary Teal**: `#0f7d70`
  - Used for: 
    - Dot color in Light Mode (DotScreenShader component).
    - "Mugshot Studio" logo text.
    - "Get Started" button background.
  - Source: User request.

## Theme Colors

### Light Mode
- **Background**: `#F4F5F5` (DotScreenShader background)
- **Dots**: `#0f7d70` (Primary Teal)
- **Dot Opacity**: `0.15`

### Dark Mode
- **Background**: `#121212`
- **Dots**: `#FFFFFF`
- **Dot Opacity**: `0.025`

## Text Colors
- **Hero Text**: `text-black` with `mix-blend-exclusion`.
- **Navbar Links**: `text-black`.
- **Sign In Button**: `text-black` (Outline).
