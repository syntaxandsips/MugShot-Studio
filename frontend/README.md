# MugShot Studio

AI-Powered HD Thumbnails in seconds.

## Overview

MugShot Studio is a professional platform designed for creators to turn their photos into high-definition, AI-crafted thumbnails.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Animations**: Framer Motion, GSAP
- **Icons**: Lucide React
- **Language**: TypeScript
